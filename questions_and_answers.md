###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `{}`

<i>Here, there is no keyword (var, let, const) before 'greetign' varialbe. JavaScript implicitly declare variable. In this case, empty object is assigned to the variable greetign in global scope.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: "12"

<i>This function receives two parameter like number and string. If any number is added to any string, number is converted to string behind the scene and concatenation is occured. Here the first parameter a receives 1 which is converted to string and concatenated to the string 2. Thus, the result is string "12".</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: Hi there, undefined

<i>Nothing is passed to this function. So, by default parameter name receives undefined. Here, undefined is placed in template string. So, the result is string.</i>

</p>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>The falsy values are 0, '', false, null, NaN, undefined. Here, the first element of nums array is 0 which is falsy value. So, the count variable will not be incremented for the element 0. That's why, final value of count is 3.</i>

</p>
</details>
